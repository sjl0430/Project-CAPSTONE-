## 1. 변수
  - BLE를 기반으로 정확한 위치를 측정하는 것은 무리가 있음
  - 위의 이유로 RSSI값을 통해서 거리값을 계산하는 것 또한 무리가 있음

## 2. 대안
  - 거리값을 직접 도출하기보다 RSSI값을 기준으로 출력하게 하기
  - 거리값을 구하는 특성 상수를 선택할 수 있게 만들기

## 3. VIEW 화면 만들기

  - 연결된 기기만 나오도록 설정

  - 아이콘의 컬러, 디자인 수정
